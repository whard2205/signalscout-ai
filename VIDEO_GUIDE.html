01 Pilih Tool Rekaman 5 menit decision

Pilih satu — jangan over-engineer. Loom paling cepat untuk demo screen + face.

ToolProConVerdict
Loom One-click record screen + webcam · auto-upload · shareable URL instant · free tier OK Watermark di free tier · max 5 menit di free ✅ Saran #1 — paling cepat
OBS Studio Free · powerful · full kontrol · no watermark · unlimited duration Setup ~15 menit · learning curve · output file lokal (manual upload) OK kalau lo butuh polish
CapCut Desktop Bagus untuk edit + captions setelah rekam · free · mobile + desktop Bukan recorder primer — pakai untuk edit setelah Loom/OBS Use as EDITOR, not RECORDER
QuickTime (Mac) Bawaan macOS · simple · zero install Cuma screen-only (no webcam combo) Backup option Mac
Rekomendasi final: Loom untuk record (15 menit). CapCut untuk edit kalau perlu trim/captions (15 menit). Total ~30 menit selesai. Skip OBS unless lo udah biasa pakainya.

02 Pre-flight Checklist 5 menit

SEMUA item harus done sebelum tekan record. Skip ini = re-record berkali-kali.

System & browser

  • Backend running: curl http://localhost:8000/health returns mode:live
  • Auto-warmup complete: tunggu 60 detik setelah backend boot. Test curl http://localhost:8000/analyze -d '{"company":"NVIDIA"}' → response < 50ms
  • Frontend running: buka http://localhost:3000 di tab clean
  • ✓
    Browser zoom 110% — text jadi lebih readable di video Loom (default 100% terlalu kecil)
  • ✓
    Full-screen browser (F11 di Chrome/Edge) — hilangkan address bar, tab clutter
  • ✓
    Tutup semua tab lain · mute notifikasi · charger nyolok
  • ✓
    Disable browser extensions yang inject UI (Honey, Grammarly, ad-blockers icon)

Environment

  • ✓
    Volume mic test — bicara, lihat waveform Loom green-level. Kalau quiet, gunakan headset/external mic
  • ✓
    Lighting di muka kalau pakai webcam — face camera ke jendela siang OR lampu meja arah muka
  • ✓
    Background clean kalau pakai webcam — wall plain atau virtual blur
  • ✓
    Hot drink + air putih dekat — voice clarity
  • ✓
    Latihan script 3× verbal sebelum hit record (next section)

Hero company untuk video

Pilih Anthropic — Claude's own company, freshest evidence (Series G ~3 months ago), active LinkedIn hiring snapshot, real AI competitors (ChatGPT, Gemini, Perplexity). Bold meta-play: "AI analyzing AI." Backup: Walmart (richest 11-role LinkedIn snapshot + Open Call program). Avoid NVIDIA for the video jika evidence rows-nya mostly >6 months old — weakens the "why NOW" thesis.

04 Common Mistakes — Yang HARUS dihindari

❌ Don't

  • "Um... so... basically..." (filler kill momentum)
  • Talking too fast (script harus muat 60 detik PELAN)
  • Click random elements yang tidak relevan
  • Scroll terlalu cepat (viewer nggak sempat baca)
  • "This is just a prototype..." (downgrade positioning)
  • Bilang "Xiaomi MiMo" atau "fallback LLM" (keep stealth)
  • Show Web Scraper "wired · not active" (test dengan hero yang pre-warmed only)
  • Cursor jumpy / shaky (gunakan trackpad/mouse smooth)
  • Notification popup tampil di video

✅ Do

  • Smile saat opening + closing
  • Pause 0.5 detik setelah bilang "deterministic"
  • Point dengan cursor — gerakan deliberate, slow
  • Sebut "Bright Data" minimal 2×
  • Sebut "Why-Now Deal Intelligence" 1-2×
  • Sebut "deterministic" 1-2×
  • End dengan static frame 2 detik
  • Re-record kalau pertama nggak smooth (mostly takes 3-5 takes)
  • Watch playback sekali sebelum publish
Pro tip dari hackathon winner: rekam 3 takes minimum. Pakai take terbaik. Jangan terima "ya udah lah". Video 60 detik akan dilihat 10-50× di LinkedIn — first impression sangat penting.
